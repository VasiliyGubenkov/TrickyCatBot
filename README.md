This is a Telegram bot that asks children's riddles. The bot was created in Python using the TeleBot library. The bot analyzes the user's messages, takes a random riddle from the database, sends it to the user in chat, and analyzes the user's response. You can test this bot using the following link (English): https://t.me/Tricky_Cat_Eng_Bot and Russian version: https://t.me/Tricky_Cat_Bot
